Getting started

Download

You can download the latest version of Metro UI Tiles on GitHub or just use links below. 

							<script src="https://cdn.rawgit.com/rtcoder/Metro-UI-Tiles-plugin/v0.6/js/metro-tiles.js"></script>
<link rel="stylesheet" href="https://cdn.rawgit.com/rtcoder/Metro-UI-Tiles-plugin/v0.6/css/metro-tiles.css">

Usage 

							<script src="metro-tiles.js"></script>
<script>
    $('div').createTile({...});
</script>

Tile configuration

Options

Name Type Required Default Description
size String Yes Define tile size:

More in size section
type String only for carousel and slider 'normal' Define type of tile:

normal (default)
slider
carousel
icon String No null URL to icon
image String No null URL to image
title String No null Title of tile
describe String No null Describe of tile
notification Numeric No null Number in bottom right corner
effect String No null Define slide and zoom effects
More in effects section
carousel Object No Object Define carousel
More in carousel section
carousel.images Array[string] only for carousel empty array Carousel images list
More in carousel section
carousel.time Numeric only for carousel 0 Carousel time interval
must be bigger than 0
More in carousel section
slider Object No Object Define slider
More in slider section
slider.images Array[string] only for slider empty array Slider images list
More in slider section
slider.time Numeric only for slider 0 Slider time interval
must be bigger than 0
More in slider section
slider.direction String only for slider null Slide direction
More in slider section
