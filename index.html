Download

You can download the latest version of Metro UI Tiles on GitHub or just use links below. 

							<script src="https://cdn.rawgit.com/rtcoder/Metro-UI-Tiles-plugin/v0.6.1/js/metro-tiles.js"></script>

    $('div').createTile({...});

Creating a Tile

Tile configuration

Options

Name Type Required Default Description
size String Yes Define tile size
type String only for carousel and slider 'normal' Define type of tile
icon URL No null Define icon
image URL No null Define image
title String No null Define title
describe String No null Define describe
notification Number No null Number in bottom right corner
effect String No null Define effects
carousel Object No Object Define carousel
carousel.images Array[string] only for carousel empty array Carousel images list
carousel.time Number only for carousel 0 Carousel time interval in miliseconds
must be bigger than 0
slider Object No Object Define slider
slider.images Array[string] only for slider empty array Slider images list
slider.time Number only for slider 0 Slider time interval in miliseconds
must be bigger than 0
slider.direction String only for slider null Slide direction

Effect

there are three types of effects: slide, zoomIn, zoomOut
slide: ['slide-right', 'slide-left', 'slide-top', 'slide-bottom']
zoomIn: ['zoomIn-right', 'zoomIn-left', 'zoomIn-top', 'zoomIn-bottom']
zoomOut: ['zoomOut-right', 'zoomOut-left', 'zoomOut-top', 'zoomOut-bottom']


Carousel


To create carousel you have to set type of tile to 'carousel'

images parameter is an array of string
time parameter allow only number and must be bigger than 0
time is an interval in miliseconds

Slider


To create slider you have to set type of tile to 'slider'

images parameter is an array of string
drection parameter allow only ['right', 'left', 'top', 'bottom', 'random']
time parameter allow only number and must be bigger than 0
time is an interval in miliseconds
